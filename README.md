# .Net Performance Tests

## Prerequisites
1. .Net 4.6.2 SDK (https://www.microsoft.com/en-us/download/details.aspx?id=53321)
2. .Net Core (https://dotnet.github.io)

## Building
From a Developer Command prompt with Powershell:

```
cd <Component>\src
.\build.ps1
```

## Running the test

```
cd <Component>\src
.\run.ps1
```
